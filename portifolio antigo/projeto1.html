PAC3

Pac3 project was created by duo team, it objective was the final score for programming laboratory class. We had to create or remake some game with python program language using the pytest library and others frameworks and libraries, if needed or whished.

We decided to remake Pacman game , but we only made 3 levels . the first easy, the second medium and the last one hard. The game also have some tricks that the player must find out to use, like the secret key for pass level that is a combination of two keys of the keyboard.

We made only using pytest library and the language python, specifically version 3.10.0 on pycharm IDE. We had to learn a lot of things in python from the begin to made work. With this project we learned the basics of python, functions, modularization, structures of projects, something about tests, had to work with git distributed version control and team work.

The greatest challenge was to make the "ghosts" work, called vilans in our game. The project doesn't look like we planed, still have problems. But, was a big boost to our learning.
